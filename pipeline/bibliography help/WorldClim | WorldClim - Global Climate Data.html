WorldClim

WorldClim is a set of global climate layers (gridded climate data) with a spatial resolution of about 1 km2. These data can be used for mapping and spatial modeling.

The current version is Version 1.4.
For this version you can get data for past, current and future climates.

A preview of Version 2 is also available (current climate only)
